# Phishing Email Detection & Awareness

## Overview
This project focuses on analyzing phishing emails and creating awareness about how to identify and prevent phishing attacks.

## Objective
- Identify phishing indicators
- Classify email risk
- Educate users on phishing threats

## Tools Used
- Email Header Analyzer
- Browser Tools
- Public phishing datasets

## Analysis
A sample phishing email was analyzed for:
- Fake sender
- Suspicious links
- Urgency tactics

## Result
The email was classified as: **PHISHING (High Risk)**

## Prevention
- Avoid clicking unknown links
- Verify sender identity
- Never share sensitive information
